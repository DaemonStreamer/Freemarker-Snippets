# FTL Extension - Functions
### [addToDatetime](/Functions/addToDatetime/)
This is a package of functions with intuitive naming to add *(or subtract)* years, months, days, hours, minutes, or seconds to any given date, time, or datetime value.

This will provide a more precise and easier way to do date/time/datetime calculations, as leap years will be calculated correctly.


### [changeTimezone](/Functions/changeTimezone/)
Change the timezone of the a datetime value.


### [csvObject](/Functions/csvObject/)
Makes it easier to create a CSV friendly JSON object.  
*Designed to be used with: "[Macro - objectToCsv](/Macros/objectToCSV/)"*
