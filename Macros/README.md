# FTL Extension - Macros


### [convertToJson](/Macros/convertToJson/)
Print any object to a JSON formatted string.


### [jsonToTable](/Macros/jsonToTable/)
Input a json object have it printed with html tables. (each tag has it own unique class id for CSS styling.)


### [objectToCsv](/Macros/objectToCSV/)
Will print a json object to CSV.  
*Designed to be used with: "[Function - csvObject](/Functions/csvObject/)"*
